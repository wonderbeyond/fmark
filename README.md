## Quick setup:

    # make it executable:
    $ chmod +x ./fmark.py

    # Copy it to some bin dir:
    $ ln -s `pwd`/fmark.py ~/bin/fmark

    # Add _bashrc to your .bashrc
    $ echo "source `pwd`/_bashrc" >> ~/.bashrc

    # Bash completion for fmark
    $ sudo ln -s `pwd`/bash_completion /etc/bash_completion.d/fmark
